## C

```C
// Increment (By one)
a++; // Postfix - Takes effect after its call
++a; // Prefix - Takes effect before its call

// Decrement (By one)
a--; //	Postfix - Takes effect after its call
--a; // Prefix - Takes effect before its call
```
