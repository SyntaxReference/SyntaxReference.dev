## C

```C
// ------------------------------------
// Basic Operators
// ------------------------------------

// Addition
a + b;

// Subtraction
a – b;

// Multiplication
a * b;

// Division & Floor Division
a / b; // If both are Integers, result will be a Integer
a / b * 1.0; // If you want an float result, you can multiply one of them to 1.0
(float) a / b; // Or, you can also use a simple type cast

// Exponent
// Not Available
// Use math.h library

// Modulus
a % b; // It returns the "remainder" after the division (modulo division)

// Increment
a++; // Increases the integer value by one

// Decrement
a--; //	Decreases the integer value by one.
```

## Python

```Python
# ------------------------------------
# Basic Operators
# ------------------------------------

# Addition
a + b

# Subtraction
a – b

# Multiplication
a * b

# Division
a / b # Result will be a Float

#Floor Division
a // b # Result will be a Integer

# Exponent
a ** b

# Modulus
a % b # It returns the "remainder" after the division (modulo division)

# Increment
# Not Available. You need to use the Addition Operator
a = a + 1

# Decrement
# Not Available. You need to use the Subtraction Operator
a = a - 1
```
